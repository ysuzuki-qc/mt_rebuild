# mt util

Utility functions for measurement tools.

## Qubit lattice

Utility functions for managing qubit posisitions and connectivitiy in RIKEN's chip convention

- Mapping from qubit index to qubit position for a given chip size
- Enumerate available CNOT pairs from alive physical qubits

## tunits typing

Wrapper of variable types `Time` and `Frequency` in `typedunits` to use them for pydantic verification
